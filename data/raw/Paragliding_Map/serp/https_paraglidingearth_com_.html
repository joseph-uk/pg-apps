A work in progress to make a simple map based collaborative database of paragliding sites, clubs and pros worldwide.

What you can find here for the moment (as of jun. 2018)
  • Fullscreen mobile-friendly map showing all pgEarth paragliding sites, pros and clubs as you pan and zoom
  • Search on flying sites or cities names
  • Sites details and pictures, clubs and pros
  • Sites maps: showing site details : main landings and parkings, but also alternate ones for sites with multiple takeoffs and/or landings...
  • Register/Log in + add + edit information about sites, clibs, pros.
    PgEarth is editable again !! Login and help us get the most up to date data !
    Former pgEarth members who had a valid email along with their account can use the "forgot your password?" feature in order to login to the new site.
  • Export to gpx: export sites in the viewport into a "gpx" file.
What you will not find here for the moment
  • 'Advanced' search sites filters
  • For sure plenty of many things i can't think of... :(
  • If you want help develop some features, welcome ! : the project source code is hosted on framagit. Feel free to join !

You like PgEarth ? YOU CAN HELP US !
  • by editing the information: log in and help keeping the information up to date. The website is efficient only if people like you keep the information correct.
  • by reporting errors: no need to be logged in : report to the moderators the information that you think should be deleted (trash icons).
  • by bringing your friends: the more the merrier. Advertise around you ! (We are not big fans of social networks, but if you are, go ahead and give us some buzz! :)
  • by translating/improving the english: information on pgearth is meant to be readen by as many people as possible, that is why we chose to publish in english : translate the data fields if you can or contact us to improve the english of the website itself.
  • by beeing a moderator: help us keep the data clean by deleting incorrect information.
  • by joining the code git: the code git is here.
  • by offering a coffee: to be frank we already have our own coffee, but that would help with hosting, domain names, bought scripts, etc.

    • Simple: only 1 or 2 waypoint(s) per flying site, corresponding to the main takeoff and landing if known
    • Detailled: extra waypoints for parkings, alternate takeoffs and landings if any
